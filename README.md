## Instalation
### Requierements
You need Python 3.6 or later to run this project. Additionally, it is necessary to install a series of dependencies. The current versions of the respective tools used are:

- numpy 1.19.1
- opencv 3.4.2
- skimage 0.16.2
- scikit_image 0.16.2
- scikit_learn 0.23.2
- matplotlib 3.3.1
- scipy 1.5.2
- tqdm 4.46.1
